# Программа обучения в боте @aist_pilot_bot

Весь теоретический и практический материал марафона выдаётся через **Telegram-бота [@aist_pilot_bot](https://t.me/aist_pilot_bot)**.

Бот автоматически присылает уроки каждый день, структурируя ваше обучение на протяжении двух недель.

---

## Как это работает

1. **Запустите бота** → [@aist_pilot_bot](https://t.me/aist_pilot_bot)
2. **Получайте уроки ежедневно** — теория утром, практика в течение дня
3. **Выполняйте задания** — бот отслеживает ваш прогресс
4. **Встречайтесь с наставником** — по расписанию четырёх встреч

---

## Блок 1: Диагностика → Первые практики

| День | Теория 📘 | Практика 🔧 |
|:----:|-----------|-------------|
| **1** | Три состояния застревания — Хаос, Тупик, Поворот. Какое твоё? | **Диагностика** — Напиши текст "Моё текущее состояние" (0,5 стр) |
| **2** | Ловушка «жизни полегче» — почему проблемы не исчезают | **Первый слот** — Выдели 15-25 мин на саморазвитие сегодня |
| **3** | Почему крутимся как белка в колесе — реактивность vs стратегия | **Трекер-помидоро** — Заполни трекер за сегодня |
| **4** | Информационный шум и потеря фокуса — внимание как ресурс | **Мимолётные заметки** — Запиши 3+ мысли по ходу дня |
| **5** | Цикл выгорания — почему рывки не работают | **Стоп-моменты** — Отследи 3+ момента ухода внимания |
| **6** | Почему советы не работают — лайфхаки vs система | **Гипотеза** — Сформулируй: "Если я буду..., то..." |
| **7** | Итоги первого блока — рефлексия и анализ | **Чек-лист блока** — Что получилось? Что сложно? |

### Результат первого блока
- Определено текущее состояние (Хаос / Тупик / Поворот)
- Запущены первые практики
- Сформулирована гипотеза до следующей встречи

## Блок 2: От практики к системе

| День | Теория 📘 | Практика 🔧 |
|:----:|-----------|-------------|
| **8** | Непродуктивные мемы — убеждения, которые мешают | **Рабочий продукт** — Создай один РП из слота |
| **9** | Героизм vs системность — маленькие шаги > рывки | **ИИ как усиление** — Оформи заметки с помощью ИИ |
| **10** | Мотивация недостаточна — система работает, мотивация кончается | **Минимальное время** — Определи своё стабильное время |
| **11** | Обучение как инженерия личности — ты проект, а не продукт | **Персональный чек-лист** — Составь список своих стоп-моментов |
| **12** | Собранность: управление вниманием — фокус как навык | **Продуктивные мемы** — Запиши 3 новых убеждения |
| **13** | Агентность: от реактивного к проактивному — действовать, не ждать | **Я как система** — Нарисуй схему: состояние → метод → РП |
| **14** | Стиль жизни созидателя — ежедневные практики | **План на месяц** — Что продолжишь делать после марафона? |

### Результат второго блока
- Освоены базовые практики
- Понимание своих мемов и стоп-моментов
- Готовый план развития после марафона

---

## Расшифровка практик

| Практика | Описание |
|----------|----------|
| **Диагностика** | Письменное описание текущего состояния жизни |
| **Слот** | Фиксированное время для саморазвития (15-25 минут) |
| **Трекер** | Ежедневная фиксация выполнения практик |
| **Мимолётные заметки** | Быстрые записи мыслей и идей по ходу дня |
| **Стоп-моменты** | Отслеживание моментов, когда внимание уходит |
| **Гипотеза** | Формулировка эксперимента: "Если я буду X, то Y" |
| **Рабочий продукт (РП)** | Конкретный результат работы (текст, схема, план) |
| **Мемы** | Убеждения, влияющие на поведение |

## Связь с наставником

Бот — это ежедневная поддержка между встречами с наставником:

```
Бот (дни 1-7)  →  Встречи с наставником  →  Бот (дни 8-14)  →  Финальная встреча
```

Наставник помогает:
- Интерпретировать результаты диагностики
- Скорректировать практики под вас
- Разобрать сложности и стоп-моменты
- Закрепить результаты марафона
