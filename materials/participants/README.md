# Материалы для участников марафона

Добро пожаловать в Марафон 2.0!

Здесь собраны все материалы для прохождения программы личного развития.
Программа состоит из 4 встреч и помогает выстроить устойчивый ритм саморазвития.

### Шаг 1. Запустите бота в Telegram
Основной способ изучения материала — через бота.

👉 **[@aist_pilot_bot](https://t.me/aist_pilot_bot)**

### Шаг 2. Определите своё состояние
Заполните опросник для диагностики текущего состояния.

👉 **[Опросник: Определение текущего состояния](00_questionnaire.md)**

Вы определите одно из трёх состояний:
- 😵 **Хаос** — внимание не принадлежит вам, постоянные отвлечения
- 🧱 **Тупик** — стабильность есть, но роста нет
- 🔁 **Поворот** — всё неплохо, но хочется большего
