# Шаблон трекера-помидоро

Используйте этот шаблон для учёта своих слотов саморазвития.

---

## Вариант 1: Простая таблица

| Дата | Время | Слот | Что делал | РП | ✓/✗ |
|------|-------|------|-----------|-----|-----|
| 10.12 | 21:00 | 🍅 | Читал методичку | Заметки 5 пунктов | ✓ |
| 11.12 | 08:30 | 🍅 | Писал текст | Черновик 0,5 стр | ✓ |
| 12.12 | — | — | Пропустил | — | ✗ |
|  |  |  |  |  |  |

## Вариант 3: Формат периода

### Период после встречи 1 (10.12 - 13.12)

**Понедельник 10.12**
- ⏰ Время: 21:00
- 🍅 Слот: 25 минут
- 📖 Что делал: Читал про стоп-моменты
- 📦 РП: Список из 3 триггеров
- ✅ Выполнено

**Вторник 11.12**
- ⏰ Время: 08:30
- 🍅 Слот: 20 минут
- 📖 Что делал: Писал про своё состояние
- 📦 РП: Текст 0,5 страницы
- ✅ Выполнено

**Среда 12.12**
- ❌ Пропущено
- 📝 Причина: Поздний созвон

```
12.12 | — | — | — | ✗
```

---

## Советы по заполнению

1. **Заполняйте сразу после слота** — не откладывайте на конец дня
2. **Будьте честными** — отмечайте пропуски, это тоже важные данные
3. **Описывайте РП конкретно** — не «думал», а «список из 5 пунктов»
4. **Не перфекционизм** — даже строка «был слот, записал мысль» лучше пустоты

---

## Что анализировать в конце периода

- Сколько слотов за период? (цель: минимум 2–3)
- В какое время получается лучше всего?
- Какие форматы РП работают?
- Какие дни/время чаще пропускаются?
- Есть ли закономерности в пропусках?

---

## Куда можно вести трекер

- **Google Sheets** — удобно для таблиц и графиков
- **Notion** — гибкие базы данных
- **Obsidian** — если уже используете для заметок
- **Тетрадь** — простой и надёжный вариант
- **Простой текстовый файл** — минимализм

Главное — **одно постоянное место** для всех слотов.
