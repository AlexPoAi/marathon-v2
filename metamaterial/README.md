# Метаматериал марафона 2.0

## Кратко о документе
- системное описание марафона и его логики;
- основа, на которой построены все материалы марафона;
- краткая версия без методических деталей.

## 🧭 Логика марафона
На каждом этапе есть:
- вход;
- метод;
- рабочий продукт.

Рабочий продукт предыдущего этапа становится входом следующего.

## 🛠️ Как устроен марафон
- Три состояния входа: Хаос, Тупик, Поворот.
- Четыре встречи: определение состояния → связка «состояние → методы → эксперимент до следующей встречи» → минимальное время и рабочие продукты → результаты и системный взгляд.
- Итог: минимальный набор практик и переход к роли Практикующего ученика.
